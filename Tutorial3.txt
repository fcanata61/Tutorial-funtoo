============================================================
Exemplo Prático: Organizando Ebuilds do Gentoo em Kits do Funtoo
============================================================

Objetivo:
- Criar um meta-repo pessoal do Funtoo usando ebuilds do Gentoo.
- Organizar os pacotes em kits e perfis para uso com emerge + ego.

------------------------------------------------------------
Estrutura de Diretórios do Meta-Repo
------------------------------------------------------------

/var/git/my-funtoo-meta-repo/
 ├── core-kit/
 │    ├── sys-libs/
 │    ├── sys-devel/
 │    ├── virtual/
 │    └── profiles/
 ├── python-kit/
 │    ├── dev-lang/
 │    └── profiles/
 ├── desktop-kit/
 │    ├── x11-base/
 │    ├── x11-libs/
 │    ├── kde-apps/
 │    ├── gnome-extra/
 │    └── profiles/
 ├── media-kit/
 │    ├── media-libs/
 │    ├── media-video/
 │    ├── media-sound/
 │    └── profiles/
 ├── net-kit/
 │    ├── net-libs/
 │    ├── net-misc/
 │    └── profiles/
 └── dev-kit/
      ├── dev-util/
      ├── dev-db/
      ├── dev-php/
      └── profiles/

------------------------------------------------------------
1) Core Kit
------------------------------------------------------------

mkdir -p /var/git/my-funtoo-meta-repo/core-kit/sys-libs
cp -r /var/tmp/gentoo-ebuilds/sys-libs/glibc* /var/git/my-funtoo-meta-repo/core-kit/sys-libs/
cp -r /var/tmp/gentoo-ebuilds/sys-libs/zlib* /var/git/my-funtoo-meta-repo/core-kit/sys-libs/

mkdir -p /var/git/my-funtoo-meta-repo/core-kit/sys-devel
cp -r /var/tmp/gentoo-ebuilds/sys-devel/gcc* /var/git/my-funtoo-meta-repo/core-kit/sys-devel/
cp -r /var/tmp/gentoo-ebuilds/sys-devel/binutils* /var/git/my-funtoo-meta-repo/core-kit/sys-devel/

mkdir -p /var/git/my-funtoo-meta-repo/core-kit/virtual
cp -r /var/tmp/gentoo-ebuilds/sys-apps/bash* /var/git/my-funtoo-meta-repo/core-kit/virtual/
cp -r /var/tmp/gentoo-ebuilds/sys-apps/coreutils* /var/git/my-funtoo-meta-repo/core-kit/virtual/

mkdir -p /var/git/my-funtoo-meta-repo/core-kit/profiles/funtoo/1.4/linux-gnu/arch/x86-64bit
echo "Core kit profile" > /var/git/my-funtoo-meta-repo/core-kit/profiles/funtoo/1.4/linux-gnu/arch/x86-64bit/README

------------------------------------------------------------
Próximos Passos
------------------------------------------------------------

1. Inicializar Git no meta-repo:
cd /var/git/my-funtoo-meta-repo
git init
git add .
git commit -m "Meta-repo inicial com kits e ebuilds adaptados do Gentoo"

2. Configurar Portage:
[my-funtoo]
location = /var/git/my-funtoo-meta-repo
masters = gentoo
auto-sync = no

3. Testar sincronização e atualização:
ego sync --config-only
emerge --sync
emerge -auDN @world
